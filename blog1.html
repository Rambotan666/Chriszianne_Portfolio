How I Started Coding

← Back to Portfolio

My journey into software development began with curiosity and a passion for technology. I was fascinated by how websites and applications worked, and I wanted to learn how to create them myself.

I started by learning basic HTML and CSS through online tutorials. As my curiosity grew, I explored Java, C#, and eventually Android development using Java in Android Studio. Each language taught me something new and strengthened my problem-solving skills.

This journey has been both challenging and exciting. Coding turned from a hobby into a serious passion that I plan to pursue as a career.
